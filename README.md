# to-do-list
A simple monochrome to do list to organize your tasks.

1) Create a tasks.txt file under the same folder as to_do_list.py 
2) Launch to_do_list.py
3) Type out your tasks and click 'Add task' to add tasks to to do list
4) Select existing task(s) and click 'Delete task' to remove unwanted or completed tasks
5) Click on 'Save list' to confirm edits
